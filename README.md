# React To-Do

> A simple to-do list in React.

## Getting started

* Install `yarn` with `npm install -g yarn`
* Clone this project
* Navigate to the project with `cd react-to-do`
* Run `yarn` to install dependencies
* Start the development server with `yarn start`
